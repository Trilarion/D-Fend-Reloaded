Ce fichier archive contient un pack de traduction en français des commandes et messages
s'affichant sous dosbox 0.72
Ce pack n'est pas compatible avec les versions précédentes de dosbox, consulter le site pour
télécharger la version appropriée.


Procédure d'installation (facile):

1. Copier le fichier "francais.lng" contenu dans l'archive vers le répertoire de dosbox 0.72
(par défaut C:\Program Files\DOSBox-0.72)

2. Ouvrir avec un éditeur de texte ou notepad le fichier "dosbox.conf" ou cliquer sur 
le raccourci "DOSBox.conf" du groupe de programmes DOSBox-0.70

3. dans la section [dosbox] modifier le paramètre language (ligne 33) en :
language=francais.lng      (ne pas mettre de ç!)

4. Enregistrer le fichier dosbox.conf et fermer l'éditeur

5. Lancer Dosbox. Il s'affiche maintenant en français!  :)

6. A l'invite de commande, pour utiliser un clavier français, taper:
keyb fr 850


nota:
Il est possible que la traduction ne concerne pas tous les messages
d'information utilisables du programme.
N'ont été traduits que les messages et les commandes disponibles en
lançant "CONFIG -writelang nom_de_fichier" sous dosbox.
