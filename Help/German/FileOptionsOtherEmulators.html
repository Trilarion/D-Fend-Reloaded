Programmeinstellungen-Dialogseite "Weitere Emulatoren"

Auf dieser Dialogseite können Sie weitere Emulatoren für alte Spiele angeben. Bei den Emulatoren muss es sich um Windowsprogramme handeltn (also nicht um Programme wie QBasic, die ihrerseits erst noch innerhalb von DOSBox ausgeführt werden müssen). In D-Fend Reloaded werden DOSBox und ScummVM als Emulationsplattformen vollständig unterstützt. Die hier angegebenen Emulatoren werden nur als Windowsprogramme unterstützt, d.h. wenn Sie ein Spiel für eine der hier angegebenen Plattformen hinzufügen, so wird intern ein Windows-Profil angelegt, in dem der angegebene Emulator aufgerufen wird und ihm die Datendatei, die das Spiel repräsentiert als Parameter übergeben.
