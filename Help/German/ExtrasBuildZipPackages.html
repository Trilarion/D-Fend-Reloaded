Zip-Pakete für mehrere Spiele erstellen

Über diese Funktion können komplette Spiele inklusive aller zugehörigen Dateien (Programmdateien, Profildatei, Datenverzeichnis, aufgezeichnete Screensgots usw.) in jeweils eine einzelne zip- oder 7z-Datei eingepackt werden. Diese Archivedateien eignen sich zur Archivierung und zum Übertragen von Spielen. Spiele, die in zip- oder 7z-Archiven gespeichert sind, können später wieder über den Menüpunkt "Zip-Datei importieren" aus dem "Datei"-Menü oder einfach in dem die Archiv-Datei auf das D-Fend Reloaded Programmfenster gezogen wird, importiert werden.

Aus dieser Liste können Sie die Spiele, für die Archive erstellt werden sollen, auswählen. Wenn Sie nur für ein einzelnes Spiel ein Archiv erstellen wollen, so können Sie dies auch über den Menüpunkt "Zip-Archiv erstellen" aus dem "Profil"-Menü.

Die in diesem Dialog erstellten Archivdateien eignen sich besonders gut für die Langzeitarchivierung (besser als die Exe-Installer-Pakete, da ein zip-Paket auch ohne D-Fend Reloaded verwendet werden kann: Außer den eigentlichen Spieledateien enthält ein Archiv auch die D-Fend Reloaded Profildatei und auch eine DOSBox conf-Datei. Dies bedeutet, dass das Spiel und seine DOSBox Konfigurationsdatei auch unabhängig von D-Fend Reloaded benutzbar bleibt.)
