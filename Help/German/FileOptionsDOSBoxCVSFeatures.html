Programmeinstellungen-Dialogseite "DOSBox CVS-Features"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Dialogseite können Sie spezielle Konfigurationsoptionen in D-Fend Reloaded aktivieren, die nur von den neueren DOSBox CVS-Builds unterstützt werden. (D.h. insbesondere, dass Sie diese Optionen nicht aktivieren sollen, wenn Sie die mit D-Fend Reloaded mitgelieferte Standard-DOSBox-Version verwenden.)

  • Mehrere Floppyimages pro Laufwerk zulassen
    Wenn diese Option aktiviert ist, können Sie im Laufwerkseditor mehrere Floppyimages pro virtuellem Laufwerk angeben. Wenn DOSBox läuft, können Sie per Strg+F4 zwischen den verschiedenen Images umschalten.
  • PhysFS Registerreiter anzeigen
    Das PhysFS (Phys Dateisystem) ermöglicht es, Zip-Dateien als DOSBox-Laufwerke einzubinden. DOSBox verwendet die Zip-Dateien nur zum Lesen und leitet jegliche Schreibbefehle in ein zusätzlich anzugebendes normales Verzeichnis um. Wenn diese Option aktiviert ist, können Sie die PhysFS-Funktion im Laufwerkseditor verwenden.
  • Sicherheitsmodus-Option aktivieren
    Wenn diese Funktion aktiviert ist, können Sie für jedes Profil auf der Laufwerke-Seite des Profileditors wählen, ob der Sicherheitsmodus aktiviert werden soll oder nicht. Wenn der Sicherheitsmodus aktiviert ist, können nach dem DOSBox-Start von der Kommandozeile aus keine weiteren Laufwerke eingebunden werden.
  • Mehr IOCTL-Optionen anbieten
    Wenn diese Option aktiviert ist, werden im Laufwerkseditor einige weitere Methoden zum Zugriff auf CD Laufwerke angeboten.
  • Grafikeinstellungen im Profileditor
    In dieser Box können Sie einige erweitere Grafikfeatures im Profileditor aktivieren, die von der Standard-DOSBox-Version nicht unterstützt werden.
  • Soundeinstellungen im Profileditor
    In dieser Box können Sie einige erweitere Soundfeatures im Profileditor aktivieren, die von der Standard-DOSBox-Version nicht unterstützt werden.
  • Druckereinstellungen im Profileditor
    In dieser Box können Sie die Einstellungen für die Druckerunterstützung im Profileditor aktivieren. Die Standard-DOSBox-Version unterstützt keine Druckeremulation, daher werden normalerweise keine Einstellungen zur Druckereulation angezeigt.
  • CPU-Einstellungen im Profileditor
    Wenn diese Option aktiviert ist, kann auf der Dialogseite CPU des Profileditors eingestellt werden, was für einen CPU-Typ DOSBox emulieren soll.
