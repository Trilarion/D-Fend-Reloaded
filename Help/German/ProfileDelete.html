Profil löschen

Wenn Sie diesen Menüpunkt anklicken, wird das gewählte Profile aus der Spieleliste gelöscht. Die Spieledatei bleiben dabei erhalten, es wird nur das Profil entfernt. Wenn Sie hingegen das gesamte Spiel mit allen dazu gehörigen Dateien (Programmdateien, Datenordner, aufgezeichnete Screenshots usw.) deinstallieren wollen, müssen Sie den Menüpunkt Deinstallieren aus dem "Profil"-Menu wählen.

Auf der Dialogseite Sicherheit des Programmeinstellungen-Dialogs können Sie wählen, ob D-Fend Reloaded beim entfernen von Profilen einen Rückfrage/Bestätigungs-Dialog anzeigen soll oder aber das Profil ohne jegliche Warnung sofort löschen soll sobald Sie den Menüpunkt anklicken.
