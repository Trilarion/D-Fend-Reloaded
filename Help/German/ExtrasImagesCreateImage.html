Imagedatei erstellen

In diesem Dialog können Sie neue, leere Disketten- und Festplattenimage-Dateien erstellen. Diese Imagedateien können dann über die Laufwerke-Dialogseite des Profileditor-Dialogs als Laufwerke in DOSBox eingebunden werden.

Diskettenimages

Auf dieser Dialogseite können Sie Diskettenimages jedes denkbaren Diskettentyps erstellen. Dafür können Sie entweder aus dem Auswahlfeld "Disketten-Typ" einen der vordefinierten Diskettentypen auswhälen oder aber über die darunter befindlichen Felder den Diskettentyp frei definieren.

  • Imagedatei im NTFS-Dateisystem komprimieren
    Wenn diese Option aktiviert ist, wird das Image als komprimierte Datei gespeichert. Dies bedeutet, dass die Datei nur soviel Platz auf dem Datenträger belegt, wie wirklich zum Speichern der in dem Image abgelegten Dateien notwenidg ist. Die leeren Sektoren innerhalb des Images belegen fast keinen Platz. Die Kompression erfolgt durch den Windows-Festplattentreiber, d.h. Sie können mit der komprimierten Datei normal arbeiten, ohne ein (De-)Kompressionsprogramm verweden zu müssen.
  • Diskettenimage bootfähig machen
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded die FreeDOS Boot-Dateien auf das Diskettenimage und erstellt einen Bootsektor auf dem Diskettenimage, welches es bootfähig macht. (Um diese Option verwenden zu können, muss FreeDOS installiert und der Pfad zu FreeDOS auf der Dialogseite FreeDOS des the Programmeinstellungen-Dialogs eingetragen sein.)
  • Die DOSBox-Tastaturbelegung auf der Bootdiskette verwenden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Tastaturtreiber auf die Bootdiskette und fügt eine Zeile in die autoexec.bat-Datei auf der Bootdiskette ein, um das Tastaturlayout, welche in der in D-Fend Reloaded gewählten Sprachdatei definiert ist, zu aktivieren.
  • Maustreiber auf der Bootdisk einbinden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Maustreiber auf die Bootdiskette und fügt eine Zeile in die autoexec.bat-Datei auf der Bootdiskette ein, um ihn beim Starten des Diskettenimages zu aktivieren.

Festplattenimages

From this dialog page you can create hard disk images of any size. D-Fend Reloaded will choose the right drive geometry automatically for you.

  • Imagedatei im NTFS-Dateisystem komprimieren
    Wenn diese Option aktiviert ist, wird das Image als komprimierte Datei gespeichert. Dies bedeutet, dass die Datei nur soviel Platz auf dem Datenträger belegt, wie wirklich zum Speichern der in dem Image abgelegten Dateien notwenidg ist. Die leeren Sektoren innerhalb des Images belegen fast keinen Platz. Die Kompression erfolgt durch den Windows-Festplattentreiber, d.h. Sie können mit der komprimierten Datei normal arbeiten, ohne ein (De-)Kompressionsprogramm verweden zu müssen.
  • Typ der Imagedatei
    Da vorformatierte Festplatten-Images in DOSBox nicht bootfähig gemacht werden können, können Sie in dieser Auswahlbox optional angeben, dass ein leeres, unformartiertes Festplatten-Image erstellt werden soll. In diesem Fall müssen Sie in DOSBox von einem bootfähigen Diskettenimage booten und das neu erststellte Festplatten-Image partitionieren und formatieren. (Diese dazu auch die Beschreibung zum Erstellen von bootfähigen Festplatten-Images in den FAQs.)
