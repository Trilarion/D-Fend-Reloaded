Profileditor-Dialogseite "ScummVM-Einstellungen"

Auf dieser Dialogseite können Sie ScummVM-spezifische Einstellungen vornehmen. Abhängig von dem auf der Dialogseite Profil gewählten Spiel, sind einige Einstellungen, die für das jeweilige Spiel nicht unterstützt werden, deaktiviert.

  • Sprache
    In diesem Auswahlfeld können Sie die Sprache, die Sie in dem Spiel verwenden wollen, auswählen.
  • Autospeicher-Intervall
    In diesem Eingabefeld können Sie die Anzahl an Sekunden zwischen zwei Autospeicher-Vorgängen festlegen.
  • Text-Geschwindigkeit
    In diesem Eingabefeld können Sie die Anzeigegeschwindigkeit für Texte einstellen.
  • Untertitel aktivieren (in Spielen mit Sprachausgabe)
    Wenn diese Option aktiviert ist, werden in Spielen, die über eine Sprachausgabe verfügen, Untertitel angezeigt. (In Spielen ohne Sprachausgabe werden die Dialogtexte unabhängig von dieser Einstellung stets angezeigt.)
  • Nachfrage vor Programmende anzeigen
    Wenn diese Option aktiviert ist, zeigt ScummVM einen Rückfragedialog vor dem Programmende an.
  • Verzeichnis für Spielstände
    In diesem Feld können Sie einstellen, ob ScummVM die Spielstände in dem Spieleverzeichnis oder in einem benutzerdefinierten Verzeichnis ablegen wollen.
  • CD-Laufwerk zum Abspielen von Musik
    In diesem Feld können Sie einstellen, welches physikalische CD-Laufwerk Sie verwenden wollen, um von ihm CD-Audio-Musik als Hintergrundmusik im Spiel abzuspielen. Das erste CD-Laufwerk im System hat die Nummer 0.
  • Nummer des zu verwendenen Joysticks
    Wenn das Spiel, das Sie konfigurieren wollen, Joystick-Eingaben unterstützt, können Sie in diesem Eingabefeld festlegen, welchen physikalischen Joystick Sie in dem Spiel verwenden wollen. Der erste mit dem Computer verbundene Joystick bekommt von ScummVM die Nummer 0.
