Program options dialog page "User interface"

There is only one option to change on this page:

  • Window size at program start
    In this dropdown box you can set up the appearance of the D-Fend Reloaded program window when the program starts. You can choose between "Default size", "Restore window size on program start", "Start minimized" and "Start maximized".
    (Default value for this option: "Default size")
