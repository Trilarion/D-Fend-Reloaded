Menu:   Profile | Delete

Delete profile

Clicking this menu item will delete the selected profile from D-Fend Reloaded. The game files will not be deleted, only the profile record will be removed. If you want to uninstall a game will all its files, you have to choose the menu item "Uninstall" from the "Profile" menu.

On the security page of the program options dialog you can choose if D-Fend Reloaded should ask you for confirmation before deleting the profile record or if the record should be deleted with out any waring if you click on the delete menu item.
