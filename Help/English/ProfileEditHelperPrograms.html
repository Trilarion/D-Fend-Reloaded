Profile editor dialog page "Helper programs"

On this page you can configurate programs that will be executed before and after the main program.

  • Execute command before main program
    The command you can enter in this box will be executed befor the main program is started. Via this command you can setup some input devices for example.
  • Run minimized (command before main program)
    If this option is activated, the command entered in the line above will be executed as a minimized window.
  • Wait before execution of the main program
    If this checkbox is checked, D-Fend Reloaded will wait for the "command before main program" program to terminate before the main program is started.
  • Execute command after the main program quits
    The command you can enter in this box will be executed after the main program gas terminated. (Attention: On Windows profiles D-Fend Reloaded might be not able to detect the correct time when the main programs quits.)
  • Run minimized (command after the main program quits)
    If this option is activated, the command entered in the line above will be executed as a minimized window.
