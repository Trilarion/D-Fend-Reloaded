Scan games folder for new games

With this function you can scan the directory defined on the directories page of the program options dialog as the games folder for new games. D-Fend Reloaded can automatically add the found games to the games list. Additionally you can setup if only games with a matching auto setup template should be added or if D-Fend Reloaded should try to add all found games. (In the second case the default template will be used for all games with no matching auto setup template.)
