Import archive file

With this program function you can import zip-file based complete profiles (including all game files) into D-Fend Reloaded. You can create zip-files of this type via the Make zip package function of the "Profile" menu.

Import via drag&drop

You can also import zip-packages by just droping them from the Explorer to the D-Fend Reloaded program window.
