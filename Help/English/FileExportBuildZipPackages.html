Build zip packages for multiple games

By using this function you can pack complete games with all of their files (program files, profile, data folder, captured screenshots) into a single zip or 7z file for archiving. You can install the games stored in a zip/7z archive at any time again via the menu item "Import zip file" from the "File" menu or by simply dragging the archive file to the D-Fend Reloaded program window.

From this list you can select multiple games for adding to the archive file. If you just want to create an archive for a single game, you can use "Make zip package" menu item from the "Profile" menu.

The archive packages are very good for long term archivating (better than the exe installers, because a zip package can also be used without D-Fend Reloaded: Besides the game files the archive file will not only contain the D-Fend Reloaded profile file but also a DOSBox conf file. This means the games and their DOSBox configuration will still be usable in the late future even without D-Fend Reloaded.)
